# 城市天气可视化统计分析系统 python706

## 项目概述

城市天气可视化统计分析系统是一个基于 Python、Flask 和 Vue 的软件，旨在全面、详细地收集城市天气情况，并通过图表和表格实现天气情况的可视化展示。系统设计有丰富的交互场景，能够直观地反映当前天气状态。

## 技术栈

- 开发工具：PyCharm、VSCode
- 技术运用：API集成、ECharts、Flask、HTML、Vue
- 数据库：MYSQL
- 相关组件：phpEnv

## 功能模块

系统主要由以下七大模块组成：

1. **登录注册模块**
   - 用户注册
   - 用户登录

2. **实时天气查询模块**
   - 查询实时天气
   - 生活指数展示
   - 数据可视化
   - 场景交互
   - 查看24小时预报
   - 未来七天预报

3. **历史天气查询模块**
   - 查询历史天气

4. **预警信息推送模块**
   - 查看预警信息

5. **用户管理模块**
   - 修改用户名
   - 修改密码

6. **实时天气地图**
   - 直观查看全国温度
   - 体感温度
   - 湿度
   - 能见度
   - 风力等级情况

7. **可视化大屏**
   - 历史天气可视化

## 系统角色

- 普通用户：可注册、登录、查询天气信息、接收预警信息、管理个人账户等。
  
## 运行环境

- Windows

## 部署步骤

1. 将 `weather_py` 项目导入 PyCharm。
2. 将 `weather_vue` 项目导入 VSCode。
3. 在 `weather_vue` 文件夹内打开命令行工具（cmd）。
4. 输入 `npm run dev` 启动 Vue-Vite 项目界面。
5. 在 PyCharm 中运行 `app.py` 文件。
